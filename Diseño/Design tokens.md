# Design tokens

Fuente: `tokens.jsx`. *"Institutional, infrastructure-grade. Sharp corners, hairline borders, minimal elevation. Palantir-adjacent."*

## Color
- **ink** (navy primario) `#112540` + escala ink80/60/40/20/10/06.
- **paper** (off-white cálido) `#FBFAF5`, paper2 `#F3F0E8`, paper3 `#E8E4DA`, white.
- Semánticos **apagados/institucionales**: success `#2F5D50`, warn `#8A6B1F`, danger `#8A2A2A`, info `#2D4F7A` (cada uno con su `*Bg`).
- **accent**: ochre `#8A6B1F`, usado **con cuentagotas**.

## Tipografía
- Sans: **Inter**. Mono: **JetBrains Mono**.
- Escala: display 56 / h1 40 / h2 28 / h3 20 / body 15 / bodySm 13 / label 12 / hairline 10 (mono+uppercase, tracking 0.16em) / mono 12.

## Forma
- Espaciado: 4/8/12/16/24/32/48/64/96.
- **Radius máximo 4px** ("no subimos más — infrastructure"): none/sm 2/md 4.
- Bordes hairline (rgba navy 10/16/32%). Elevación mínima (flat / e1 / e2).
