# Moat (ventaja competitiva real)

El matching es una commodity y no genera moat. El verdadero moat es otro:

1. **Datos institucionales.** Cada práctica registrada genera datos sobre cómo funciona realmente el sistema educación↔empleo. Con el tiempo, dataset imposible de replicar.
2. **Switching cost estructural.** 18–24 meses de [[Capa - Event log|event log]] = memoria institucional. Migrar = perder la trazabilidad histórica. Nadie lo hace.
3. **[[Capa - Policy layer|Policy layer]] como barrera.** Cuanto más personalizado, más difícil de reemplazar.
4. **Red entre instituciones.** Con dos universidades dentro, compartir prácticas entre ellas se vuelve trivial. No existe en ninguna plataforma actual ([[Tesis 1 - Relationship Expansion Engine]]).
