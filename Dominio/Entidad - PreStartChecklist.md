# PreStartChecklist · _Checklist de inicio_

Comprobación de readiness **rojo/amarillo/verde** antes de que el alumno empiece. "Una práctica está lista cuando todas las condiciones para empezar están cerradas", no cuando alguien dice "aceptado".

|---|---|
